1. ✅ motherboard.glb - Papan sirkuit utama
2. ⏳ processor.glb - CPU/Processor
3. ⏳ ram.glb - Memory RAM
4. ⏳ vga.glb - Graphics Card (GPU)
5. ⏳ psu.glb - Power Supply Unit
6. ⏳ ssd.glb - Solid State Drive
7. ⏳ hdd.glb - Hard Disk Drive
8. ⏳ heatsink_fan.glb - CPU Heatsink & Fan
9. ⏳ cpu_socket.glb - CPU Socket
10. ⏳ chipset.glb - Motherboard Chipset
11. ⏳ cmos_battery.glb - CMOS Battery
12. ⏳ bios_chip.glb - BIOS Chip
13. ⏳ pcie_slot.glb - PCIe Expansion Slot
14. ⏳ vrm.glb - Voltage Regulator Module
15. ⏳ capacitor_resistor.glb - Capacitor & Resistor
16. ⏳ casing.glb - PC Case/Casing
17. ⏳ case_fan.glb - Case Fan
18. ⏳ cables.glb - Power & Data Cables
19. ⏳ frontpanel.glb - Front Panel IO

SUMBER MODEL 3D:

FREE MODEL WEBSITES:
- Sketchfab (https://sketchfab.com) - RECOMMENDED
- Free3D (https://free3d.com)
- CGTrader Free (https://www.cgtrader.com)
- Turbosquid Free (https://www.turbosquid.com)

SEARCH TIPS:
- Ketik: "CPU 3D model" atau "Processor GLB"
- Filter by: Free, GLB format
- Cek rating dan download count
- Preferensikan model berkualitas tinggi

CONVERSION:
Jika model bukan GLB format:
- Use online converter: https://convertio.co
- Or use Blender untuk convert ke GLB

OPTIMIZATION:
- Target size: < 10MB per model
- Gunakan TinyGLTF untuk compress
- Remove unnecessary geometry
- Optimize textures

PLACEMENT:
- Copy semua .glb files ke folder: assets/
- Nama file harus match dengan ar.html references

TESTING:
- View GLB files: https://www.babylonjs-playground.com/
- Validate files: https://github.khronos.org/glTF-Sample-Models/viewer/
