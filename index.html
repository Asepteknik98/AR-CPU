Tentang Aplikasi

AR CPU - Augmented Reality untuk Edukasi Teknologi

Aplikasi ini dirancang untuk memberikan pengalaman belajar interaktif tentang komponen-komponen komputer menggunakan teknologi Augmented Reality.

Dengan menggunakan teknologi AR.js dan A-Frame, pengguna dapat melihat model 3D dari berbagai komponen komputer dalam lingkungan nyata mereka.

Teknologi yang Digunakan:

  • A-Frame - Framework WebXR
  • AR.js - Library Augmented Reality
  • Three.js - 3D Graphics Library
  • Babylon.js - Game Engine

Cara Menggunakan

1. Cetak marker dari folder marker/

2. Buka halaman AR

3. Arahkan kamera ke marker

4. Lihat model 3D muncul di layar
